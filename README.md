# All-in-One Web App 📱

A mobile-style web app with **4 built-in tools** — no installs, no sign-ups, no backend required.

![App Preview](https://img.shields.io/badge/Status-Live-brightgreen) ![HTML](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

---

## Features

| Module | Description |
|--------|-------------|
| ✅ **Tasks** | Add tasks with Work / Personal / Urgent tags, check them off, delete them |
| 📝 **Notes** | Save titled notes, review and delete instantly |
| ₹ **Budget** | Track income & expenses by category with a live spending bar (Indian Rupees) |
| 🎮 **Game** | 8-pair memory card matching game for a quick mental break |

## How to Run

### Option 1 — Open directly
Double-click `index.html` to open it in your browser. That's it!

### Option 2 — GitHub Pages (recommended)
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your app will be live at `https://yourusername.github.io/repo-name`

## Customisation

- Change the app name in the `<title>` tag and `.app-title` div
- Update the currency symbol (search `₹`) to your local currency
- Add more task tags in the `<select id="t-tag">` dropdown
- Modify colors by changing `#534AB7` (primary purple) throughout the CSS
